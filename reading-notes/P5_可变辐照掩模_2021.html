可变辐照掩模（VIM）对电子元件聚焦红外光焊接系统的影响

The effect of variable irradiation mask in Focused Infrared Light Soldering Systems for electronic components

Citlalli Anguiano, Kiyoshi Natzu, Marco Félix, et al. - Optik 2021, 242: 167298, Universidad Autónoma de Baja California / CICESE

EI

DOI: 10.1016/j.ijleo.2021.167298 | 出版: 2021-05-24

关键词：聚焦红外光（Focused Infrared Light）; 可变辐照掩模（Variable Irradiation Mask）; 焊接系统（Soldering System）; 表面贴装技术（SMT）; 非相干光（Incoherent Light）

摘要与研究定位

表面贴装技术（SMT（Surface Mount Technology））让印刷电路板（PCB（Printed Circuit Board））能够容纳更多形状、尺寸各异的电子元件，并且开始把光学模块（滤波器、光电探测器、光纤等）集成进来——这就是所谓的板上光学（Optics on Board）。这类新型光子元件对焊接过程更敏感，需要一种非接触、面积可控、且不损伤元件的焊接方法。

本文在一套聚焦红外光焊接系统（FILSS，Focused Infrared Light Soldering System）的输出端插入一块可变辐照掩模（VIM，Variable Irradiation Mask），以获得一个特定尺寸与形状的光分布，并将仿真得到的辐照度分布与实验测得的热分布做对照，被测对象是一种典型封装——球栅阵列（BGA，Ball Grid Array）。

Note

论文的核心发现只有一句话，但很关键：在 VIM 遮挡边界处，得到的强度恰好约为最大热分布强度的 50%。这个"50%"不是巧合，而是不相干光（incoherent light）在直边（straight edge）处衍射的固有物理结果——边界处光强总是从"全亮"到"全暗"经过一个 50% 的过渡点。作者正是把这一衍射规律当作可预测的"设计工具"来使用：光强分布 → 视场因子 → 传热量 → 温度，四者一一对应，从而让焊接热场变得可仿真、可预测。

1. 引言：为什么需要新焊接法

电子工业的趋势是把光学模块、滤波器、光电探测器和光纤集成进电子元件，以提升集成电路（IC）内部的数据传输速度、增大系统带宽。技术演进路线可以概括为：

封装技术演进对比

技术放置方式特点与局限
THT（通孔技术）元件引脚穿过 PCB 打孔插装系统笨重、只能用板的一面、引脚数受限、传输速率低
SMT（表面贴装）元件直接贴装在 PCB 表面两面均可用、引脚数量大增、系统紧凑、处理速率高
BGA（球栅阵列）引脚（焊球）分布在 IC 底部SMT 中最流行的封装之一，广泛用于 FPGA、微处理器、DSP
板上光学在 PCB 内集成光子元件用"光"提速，带宽更高，但对焊接热损伤更敏感

由于光子与半导体材料的组合，新元件沿集成电路的热分布可能不再均匀。这正是问题所在：焊接必须遵循回流焊工艺（reflow soldering），它包含四个阶段——预热（pre-heat）、浸润（soak）、回流（reflow）、冷却（cooling）。

Note

回流焊的温度窗口是理解全文的关键标尺：

  • 预热 + 浸润阶段：温度须保持 $\mathrm{T} < 217\,^{\circ}\mathrm{C}$，目的是避免元件遭受热冲击（thermal shock）。
  • 回流阶段：温度落在 $217\,^{\circ}\mathrm{C} \le \mathrm{T} \le 260\,^{\circ}\mathrm{C}$，这个区间才能让无铅焊膏发生固→液相变，形成真正的焊点。

记住这两个数字：只有把温度推到 217 °C 以上，焊接才真正"发生"。后文会看到，本文单靠 VIM+FILSS 只能达到约 163–170 °C（仍在预热区），需要加镀增透膜才能进入回流区。

1.1 提出的方法（VIM + FILSS）

所提方法是在 FILSS 与待焊目标之间放置一块 VIM，从而在光子 SMT 元件上得到一个特定面积的光与热分布（见图 2）。FILSS 使用带椭圆金反射镜的石英钨卤灯（QTH lamp）作为红外源，波长覆盖从紫外到红外的宽谱段；其光路基于一套成像匀光（imaging homogenizer）系统。

由于 SMT 元件靠热传导焊接，且不能被任何方向的力挤压（否则会导致元件错位 misalignment），VIM 不能接触元件。为此作者分析了两种放置情形：

  • 情形一（近距离 Dz1）：VIM 靠近目标但不接触，x 轴需相应偏移 Dx1；
  • 情形二（远距离 Dz2）：VIM 远离目标，x 轴偏移 Dx2。

2. SMT 组件热分布理论

错位与冷焊（cold solder）会损害整个系统的功能，因此焊接系统产生的温度与传热必须均匀，并达到合金熔点才能形成良好焊点。分析这类系统（如 FILSS）的三条途径是：热传递、红外辐射、热传导。

辐射传热 — 式 (1)

对 FILSS，辐射传热由下式给出：

$$Q_{IR} = V \cdot \varepsilon_{s} \cdot \Gamma_{i} \cdot \Gamma_{OE} \cdot \alpha_{SMT} \cdot \sigma \cdot (T_{S}^{4} - T_{SMT}^{4}) \tag{1}$$

解读：这是斯特藩–玻尔兹曼定律（Stefan–Boltzmann law）的工程化写法。核心是 $(T_S^4 - T_{SMT}^4)$：辐射传热正比于灯源温度与元件温度的四次方之差——所以温度稍有提升，辐射量急剧增大。前面一串系数（$V,\varepsilon_s,\Gamma_i,\Gamma_{OE},\alpha_{SMT}$）都是 0–1 之间的"打折因子"，逐级衰减真正到达元件的能量。

式 (1) 各符号含义

符号含义取值/单位
$Q_{IR}$辐射传热量W/cm²
$V$系统几何视场因子（geometric view factor）0–1
$\varepsilon_s$红外源发射率（emissivity）0–1
$\Gamma_i$光学元件内部透射率0–1
$\Gamma_{OE}$各玻璃-空气界面累积透射系数0–1
$\alpha_{SMT}$SMT 元件吸收率（absorptivity）0–1
$\sigma$斯特藩–玻尔兹曼常数$5.67\times10^{-12}$ W/cm²/K⁴
$T_S,\ T_{SMT}$灯源、SMT 元件温度K
Note

为什么作者要把 VIM 的衍射效应"塞进"视场因子 $V$？因为 $V$ 是唯一一个描述"有多少辐射几何上能到达目标"的因子。VIM 遮挡了一部分光路，在边界处衍射使强度降到 50%，等价于把该处的 $V$ 从 1 打折到 0.5。这样无需重写整个物理模型，只要替换 $V$，式 (1) 就能预测 VIM 边界的传热量——这是本文把光学衍射与热学传热连接起来的桥梁。

传导传热 — 式 (2)

$$Q_{C} = K \cdot A \frac{(T_{1} - T_{2})}{\Delta x} \tag{2}$$

解读：这是傅里叶导热定律（Fourier's law）。热量在元件厚度方向上传导：$T_1$（顶面）与 $T_2$（底面）的温差越大、材料越薄（$\Delta x$ 小）、导热系数 $K$ 越高、截面积 $A$ 越大，传导的热量 $Q_C$ 就越多。物理链条是：红外辐射先打到元件顶面（式 1），再靠传导（式 2）把热量送到底部焊球把焊料熔化。

式 (2) 各符号含义

符号含义单位
$Q_C$传导传热量W
$K$热导率（thermal conductivity）W/cm-°C
$A$横截面积cm²
$T_1,\ T_2$元件顶面、底面温度°C
$\Delta x$体材料厚度cm

直边衍射 — 式 (3)（全文核心）

由于使用白炽灯作为红外源，且 VIM 采用直边（straight edge）设计，因此需要分析不相干光在直边处的衍射。此时波的叠加产生衍射图样，一些光线以不同角度混乱地进入 VIM 所限定的区域。直边不相干光衍射的强度由下式给出：

$$I_{(r)} = \frac{1}{2} + \frac{1}{\pi} Si_{(r)} - \frac{(1 - \cos(r))}{r} \tag{3}$$

解读：其中 $r = 2a\pi y /[(1+m)\lambda f]$，$a$ 是入瞳（有效孔径）直径，$m$ 是光学系统放大率，$\lambda$ 是波长，$f$ 是焦距，$y$ 是直边位置，$Si(r)$ 是正弦积分函数。关键在于：当 $y=0$（正好在直边边界）时，$Si(0)=0$、$\cos(0)=1$，公式退化为 $I=\tfrac{1}{2}$——边界处强度恰好是总强度的 50%。这就是全文那个"50%"的数学来源。

Note

这里必须区分两种衍射：相干光（如激光）在直边处会产生明暗相间的振荡条纹（菲涅耳条纹）；而本文用的是不相干白炽光，各波长、各相位随机叠加，振荡被"抹平"，只留下一条平滑的 S 形过渡曲线——强度从边界外的 0 平滑爬升到边界内的 100%，正中间穿过 50%。正因为这条曲线平滑且可预测，作者才能把"边界 = 50% 强度"当作一条可靠的工程设计规则。

总传热 — 式 (4) 与 (5)

由于直边衍射造成 50% 的强度衰减，进而造成辐射传热减少。把式 (3) 得到的强度当作视场因子 $V$ 代入式 (1)，得到总辐射传热：

$$Q_{TIR} = \Sigma Q_{DIR} + Q_{IR} \tag{4}$$

解读：$Q_{TIR}$ 是到达外表面的总辐射传热；$Q_{DIR}$ 是 VIM 衍射造成的辐射传热"累积衰减量"；$Q_{IR}$ 是来自 FILSS 的直接辐射（式 1）。直观理解：总热量 = 直接辐射 + 被衍射削弱的那部分，后者正是 VIM 用来"雕刻"边界热场的部分。

$$Q_{TC} = \Sigma Q_{DC} + Q_{C} \tag{5}$$

解读：传导侧的对应关系。$Q_{TC}$ 是总传导传热；$Q_{DC}$ 是 VIM 衍射带来的传导传热衰减量；$Q_C$ 是 FILSS 的直接传导（式 2）。本分析忽略了元件四周环境的传热——这是一个简化假设，读文献时需留意其适用边界。

本节要点速查

公式一句话解释
式 (1) 辐射灯源以 $T^4$ 律辐射，经一串 0–1 系数衰减后到达元件顶面
式 (2) 传导热量按傅里叶定律穿过元件厚度传到底部焊球
式 (3) 衍射不相干光在直边处平滑过渡，边界强度恒为 50%（全文核心）
式 (4)(5) 总量总传热 = 直接项 + VIM 衍射造成的衰减项

3. 仿真与实验方法

辐照度仿真在 Zemax 软件中用非序列模式（non-sequential mode）完成。光源建模为椭圆反射镜 + 灯丝，光功率 69.8 W。光学元件的透镜采用紫外熔石英（UV Fused Silica）与 B270 玻璃——因其热膨胀系数低、光学质量优良，且在紫外到近红外区有良好透过窗口。VIM 作为"挡光元件"放在 FILSS 与目标（探测器）之间，以获得特定面积的光分布。

Note

非序列模式是 Zemax 处理这类系统的关键：普通"序列模式"假设光线按固定顺序穿过每个面，适合成像镜头；但本系统里光线会被 VIM 遮挡、散射、以任意角度乱窜，顺序不确定，因此必须用非序列模式做蒙特卡洛式光线追迹，才能真实统计到达目标面的辐照度分布。

实验方面：FILSS 架设在光学平台上，光源为发射谱覆盖紫外到红外的钨卤灯；用热成像相机（thermography camera）测量目标处的热分布，VIM 置于末级透镜与目标之间。

一条重要的几何标定关系：为在 x 轴维持特定工作区，挡光块需随元件间距重定位——在 z 轴距目标 10 mm 时，x 轴需从光轴偏移约 2 mm；该关系适用于目标尺寸 20–30 mm 的范围。

4. 结果与分析

带成像匀光系统的 FILSS 在整个目标上产生均匀辐照；一旦插入 VIM，就在输出端得到特定面积的光/热分布。作者分析两种情形：VIM 距目标 20 mm（近）与 30 mm（远）。

Note

热分布与辐照度分布的形状相同，但要注意一个关键区别——对比度并不相同：

  • 辐照度（表 1）：底部/峰值 = 11.08/24.46 ≈ 45%，10.75/24.39 ≈ 44% → 确实约为一半，严格印证直边衍射的"50% 边界"规律。
  • 温度（表 2）：底部/峰值 = 142.5/163 ≈ 87%，143.78/170.11 ≈ 85% → 并非一半。

这两种百分比并非矛盾，而是斯特藩-玻尔兹曼定律 T⁴ 关系的必然体现：辐照度降为 50% → 表面在辐射平衡下 $(0.5)^{1/4} \approx 0.84$，温度自然降到峰值的 ~84%。实测 85–87% 略高，多余的几个百分点来自横向热传导（中心热量流向边缘）。反向验证亦然：$0.85^4 \approx 0.52$——VIM 边界向空间净辐射的热量 ≈ 50%，与入射辐照度严格平衡。原文直接称温度也是"50%"表述不够清晰（省略了 T⁴ 推导），但数据背后物理完整自洽。远距离峰值略高（170 > 163 °C），作者解释为远距离时更多光线抵达被遮挡区域。

Note

致命短板与解决方案：本系统峰值温度只有 163–170 °C，落在预回流区（pre-reflow zone），达不到 217 °C 的熔焊门槛。作者的对策是给 FILSS 每个光学界面镀增透膜（AR coating，Anti-Reflection coating）——减少界面反射损失可让温度提升多达 40%，达到 228.2–238.1 °C，从而真正进入回流区、完成焊接。但要注意：即便镀了 AR 膜，VIM 边界位置（50% 强度处）的温度仍只有约 200 °C，仍属预热/浸润区——这意味着 VIM 遮挡边缘天然不会被焊上，恰好起到"保护非焊接区"的作用。

5. 结论

作者分析了 VIM 置于 BGA 光子 SMT 元件近距离与远距离时，FILSS 输出光束的热与辐照度分布，主要结论：

  1. Zemax 仿真的辐照度分布与实验测得的热分布吻合良好，证明"光分布可预测热分布"。
  2. 近/远距离（20/30 mm）的峰值辐照度与顶宽相近，主要差别在底宽：远距离底宽更大（光斑更展宽）。
  3. 远距离时遮挡区温度更高（更多射线到达），近距离时射线密度与温度更低。
  4. 峰值温度：近距离 163 °C，远距离 170.11 °C；VIM 位置底部温度 142.5 / 143.78 °C（为峰值的 ~85–87%）。入射辐照度降为 50% 后经 T⁴ 转化 → 温度自然降到 ~84%，实测一致；$0.85^4 \approx 0.52$ 证实辐射平衡吻合。
  5. 需给光学元件镀 AR 膜把温度提升到 228.2–238.1 °C，才足以进入回流阶段完成良好焊接。

VIM 带来 50% 的强度/热量效应"显著地"影响了光子元件所能达到的温度，进而直接决定能否正确焊接。本工作揭示了辐照度与热分布之间的定量关系，可为开发采用 VIM、并考虑光路中不同位置及不相干衍射效应的新型焊接系统提供参考。

核心要点总结

  1. 问题：板上光学时代，光子 SMT 元件需要"非接触 + 面积可控 + 不损伤"的定向焊接。
  2. 方法：在聚焦红外焊接系统（FILSS）输出端插入可变辐照掩模（VIM），雕刻出特定面积的光/热场。
  3. 核心物理：不相干光在直边处衍射 → 边界强度平滑过渡且恒为 50%（式 3），把它当作视场因子 $V$ 代入斯特藩–玻尔兹曼式，就打通了"光学 → 传热"。
  4. 验证与物理自洽：Zemax 仿真辐照度与热成像实验的分布形状高度一致。辐照度降为 50% → 经 T⁴ 转化 → 温度自然降为 ~84%，实测 ~85–87% 与之吻合；$0.85^4 \approx 0.52$ 证实辐射平衡闭环。原文直接将温度也表述为 50% 不够清晰，但数据无矛盾。
  5. 距离效应：近/远距离峰值相近，差别在底宽（远距离光斑更展宽、遮挡区温度略高）。
  6. 局限与出路：裸系统仅 163–170 °C（预回流区），需镀 AR 膜提温至约 228–238 °C 才能真正回流焊接。
